---- Module with ESP32

Overview

The MAX17043 module is a battery fuel gauge that provides...

  • ESP32 microcontroller
  • MAX17043 battery fuel gauge module
  • Jumper wires
  • Breadboard (optional)

Wiring Connections

For this implementation, ...

----- ESP32
VCC N/A N/A
GND N/A N/A
SDA GPIO 32 N/A
SCL GPIO 33 N/A
B+ N/A B+
B- N/A B-
