LoRa Modules with ESP32

Overview

LoRa (Long Range) modules are wireless communication devices designed for low-power data transmission at long ranges, making them ideal for Internet of Things (IoT) applications. These modules can communicate over several kilometers in open environments and hundreds of meters in urban areas. The LoRa modules used in this implementation are RFM95W transceivers, which are based on the sx1276 chip. The modules use this chip to operate on 915MHz, which is the legal frequency band in North America. There are other frequency bands and regulations depending on the region. For example, Asia uses 433 MHz and Europe uses 868MHz. Make sure to choose the appropriate frequency band based on your regionâ€™s regulations.

Hardware Required

  • ESP32 microcontroller
  • 2 LoRa modules (RFM95W)
  • External antenna (if required)
  • Jumper wires
  • Breadboard (optional)

Wiring Connections

This implementation will use the SPI interface for communication between the ESP32 and the LoRa modules. The default SPI pins on the ESP32 are GPIO 18 (SCK), GPIO 23 (MOSI), and GPIO 19 (MISO). Additionally, the LoRa modules require the NSS, RST, and DI0 pins to be connected to the ESP32. The connections are as follows:

LoRa ESP32
VCC 3v3
GND GND
SCK GPIO 18
MOSI GPIO 23
MISO GPIO 19
NSS GPIO 4
RST GPIO 5
DI0 GPIO 2

LoRa Receiver Example Code

/* LoRa receiver code with ESP32 */
#include <LoRa.h>
#include <SPI.h>

#define NSS 4
#define RST 5
#define DI0 2

String LoRaData;

void setup() {
    Serial.begin(115200);
    LoRa.setPins(NSS, RST, DI0); // Set the pins for LoRa module

    while (!LoRa.begin(915E6)) { // Change the frequency based on your region
        Serial.println(".");
        delay(500);
    }
    LoRa.setSyncWord(0xF1); // Set the sync word to match with the transmitter
    Serial.println("LoRa Initializing Successful!");
}

void loop() {
    int packetSize = LoRa.parsePacket(); // Check if a packet is received
    if (packetSize) {
        Serial.print("Received packet: ");
        while (LoRa.available()) {
            LoRaData = LoRa.readString(); // Read the data
        }
        Serial.println(LoRaData); // Print the received data onto the serial monitor
    }
}
